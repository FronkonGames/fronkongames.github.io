Configuration for Builtin

To use ‘One Bit’ for Builtin render (Unity default render) you just have to add the ‘One Bit (Builtin)’ component to the camera of your scene.

 

Configuration for URP

To use ‘One Bit’ with URP render (Universal Render Pipeline), the first thing you should do is install the ‘Universal RP’ package version 10.8.1 or higher. For more information on how to do this, you can follow this official tutorial.

Once the ‘Universal RP’ package can be installed correctly, you can now install ‘One Bit’. Once installed, locate the file ‘Universal Render Pipeline Asset’, you can find it in the window ‘Project Settings’ > ‘Graphics’:

 

Select it and locate the file that appears in ‘Render List’:

 

Now you can add ‘One Bit Urp’ by clicking on ‘Add Renderer Feature’:
